# Kotmix
XAMPP for Linux 及び Docker 用の簡易アーカイブビュアー

# 使い方
* Kotmix.php を htdocs に置きます。
* JPG や PNG が入ったフォルダを .tar.gz 圧縮して /var/tmp に置きます。
* ブラウザで http://localhost/Kotmix.php にアクセス。
* ブラウザに画像が表示されます。画像は下に並びます。

# 用途例
* かわいい猫画像の圧縮ファイルを堪能する。
* 自作イラストの圧縮ファイルを閲覧する。
